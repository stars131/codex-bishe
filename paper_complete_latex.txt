\begin{abstract}
Network intrusion detection faces two major challenges: heterogeneous feature sources and severe class imbalance. This paper implements and evaluates a multi-source fusion framework for five-class intrusion detection (\texttt{dos}, \texttt{normal}, \texttt{probe}, \texttt{r2l}, \texttt{u2r}). We compare a single-source baseline with four fusion strategies: concatenation, attention, gated fusion, and cross-attention fusion. Experiments are conducted on a processed dataset split of 101,909/14,559/29,118 (train/validation/test). Results show that \texttt{fusion\_gated} achieves the best overall performance (Accuracy 0.998042, Weighted-F1 0.998019, Macro-F1 0.887364), and improves Macro-F1 by 0.031375 over the single-source baseline. The findings indicate that gated fusion provides a better trade-off between accuracy, minority-class robustness, and model complexity.
\end{abstract}

\section{Introduction}
Intrusion detection is a core component of network security operations. Traditional ML-based methods usually rely on a single feature space, which may miss complementary information across different traffic views. Multi-source fusion offers a practical way to integrate heterogeneous signals and improve generalization under class imbalance.  

This work focuses on:  
\begin{itemize}
    \item building a reproducible multi-source deep learning pipeline;
    \item comparing representative fusion mechanisms under the same training protocol;
    \item quantifying the gain of fusion models over a single-source baseline.
\end{itemize}

\section{Method}
\subsection{Problem Setting}
Given two source features $\mathbf{x}^{(1)} \in \mathbb{R}^{d_1}$ and $\mathbf{x}^{(2)} \in \mathbb{R}^{d_2}$, the goal is to predict attack class $y \in \{1,\dots,C\}$ where $C=5$.

\subsection{Model Families}
\textbf{Single-source baseline:} \texttt{single\_source\_mlp} uses concatenated features as one branch and trains a standard MLP classifier.  

\textbf{Fusion models:}
\begin{itemize}
    \item \texttt{fusion\_concat}: direct feature concatenation before classification;
    \item \texttt{fusion\_attention}: attention weights on source representations;
    \item \texttt{fusion\_gated}: learnable gate to adaptively control source contribution;
    \item \texttt{fusion\_cross}: bidirectional cross-attention between source embeddings.
\end{itemize}

\section{Experimental Setup}
\subsection{Data}
Processed dataset statistics:
\begin{itemize}
    \item Train: 101,909 samples
    \item Validation: 14,559 samples
    \item Test: 29,118 samples
    \item Source dimensions: $d_1=22$, $d_2=19$
    \item Classes: \texttt{dos}, \texttt{normal}, \texttt{probe}, \texttt{r2l}, \texttt{u2r}
\end{itemize}

\subsection{Training Configuration}
All models are trained with a unified protocol:
\begin{itemize}
    \item Optimizer: AdamW (lr=0.001, weight decay=$10^{-4}$)
    \item Scheduler: cosine with warmup=2 epochs
    \item Epochs: 8
    \item Batch size: 256
    \item Mixed precision: enabled
    \item Device: NVIDIA GeForce RTX 4060 Laptop GPU
\end{itemize}

\subsection{Metrics}
We report Accuracy, Weighted-F1, Macro-F1, and parameter count. Macro-F1 is emphasized for class-imbalance sensitivity.

\subsection{Discussion}
Compared with \texttt{single\_source\_mlp}, \texttt{fusion\_gated} improves:
\begin{itemize}
    \item Accuracy: +0.000378
    \item Weighted-F1: +0.000327
    \item Macro-F1: +0.031375
\end{itemize}
The Macro-F1 gain is substantially larger than the gain in Accuracy, indicating better robustness on minority classes. This is consistent with the class-wise result where \texttt{u2r} improves from 0.380952 to 0.533333.

\section{Conclusion}
This study demonstrates that multi-source fusion is beneficial for intrusion detection under class imbalance. Among tested methods, gated fusion provides the best balance between performance and computational cost. Future work should include (1) longer training and repeated runs for confidence intervals, (2) domain-shift evaluation with additional institutional datasets, and (3) threshold/calibration strategies tailored to rare attack categories.
