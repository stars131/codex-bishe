% ================= 中文摘要 =================
\chapter*{摘\quad 要}
针对网络入侵检测中异构特征利用不足与类别不平衡问题，本文构建并实现了一个多源融合深度学习框架。以五分类任务（dos、normal、probe、r2l、u2r）为目标，比较单源基线与四种融合策略（concat、attention、gated、cross）。实验结果表明，gated 融合在测试集上取得最优性能：Accuracy=0.998042，Weighted-F1=0.998019，Macro-F1=0.887364；相较单源基线，Macro-F1 提升 0.031375。结果说明门控融合在保持整体精度的同时，对少数类识别更具优势。

\noindent\textbf{关键词：} 网络入侵检测；多源数据融合；深度学习；类别不平衡；门控机制

% ================= 英文摘要 =================
\chapter*{Abstract}
To address heterogeneous feature utilization and class imbalance in network intrusion detection, this thesis implements a multi-source deep learning framework for five-class classification (\texttt{dos}, \texttt{normal}, \texttt{probe}, \texttt{r2l}, \texttt{u2r}). We compare a single-source baseline with four fusion strategies: concat, attention, gated, and cross-attention fusion. Experimental results show that gated fusion achieves the best performance (Accuracy 0.998042, Weighted-F1 0.998019, Macro-F1 0.887364), improving Macro-F1 by 0.031375 over the baseline.

\noindent\textbf{Key Words:} Intrusion Detection; Multi-Source Fusion; Deep Learning; Class Imbalance; Gated Fusion

\chapter{绪论}
\section{研究背景与意义}
网络入侵检测是网络安全防护体系中的关键环节。传统方法通常基于单一特征空间建模，难以充分利用不同来源信息之间的互补性。在复杂攻击场景下，类别分布长尾与特征异构并存，导致模型对少数类识别能力不足。因此，研究多源融合方法具有现实意义。

\section{国内外研究现状}
现有研究主要包括：基于机器学习的检测方法、基于深度学习的检测方法，以及面向多模态/多源信息融合的检测方法。总体趋势是从手工特征向端到端表示学习转移，但在跨源建模和少数类稳健性方面仍存在改进空间。

\section{本文主要工作}
\begin{itemize}
    \item 设计并实现多源融合入侵检测框架；
    \item 在统一训练协议下系统比较五类模型；
    \item 从整体指标与分类别指标分析融合策略优势。
\end{itemize}

\chapter{相关理论与方法}
\section{问题定义}
给定两路输入特征 $\mathbf{x}^{(1)} \in \mathbb{R}^{d_1}$ 与 $\mathbf{x}^{(2)} \in \mathbb{R}^{d_2}$，预测类别标签
\begin{equation}
 y \in \{1,\dots,C\}, \quad C=5.
\end{equation}

\section{模型结构}
\subsection{单源基线模型}
\texttt{single\_source\_mlp} 使用拼接后的单分支输入进行分类。

\subsection{多源融合模型}
\begin{itemize}
    \item \texttt{fusion\_concat}：直接拼接融合；
    \item \texttt{fusion\_attention}：注意力分配源权重；
    \item \texttt{fusion\_gated}：门控控制源信息贡献；
    \item \texttt{fusion\_cross}：交叉注意力建模源间交互。
\end{itemize}

\chapter{实验设计}
\section{数据集与划分}
实验使用当前项目已处理数据，训练/验证/测试样本分别为 101,909 / 14,559 / 29,118，输入维度为 $d_1=22$、$d_2=19$，类别为 dos、normal、probe、r2l、u2r。

\section{实验环境}
\begin{itemize}
    \item GPU：NVIDIA GeForce RTX 4060 Laptop GPU
    \item 深度学习框架：PyTorch
    \item 训练策略：Mixed Precision
\end{itemize}

\section{训练配置}
\begin{itemize}
    \item Optimizer：AdamW（lr=0.001，weight decay=$10^{-4}$）
    \item Scheduler：Cosine + Warmup(2 epochs)
    \item Epochs：8
    \item Batch size：256
\end{itemize}

\section{评价指标}
采用 Accuracy、Weighted-F1、Macro-F1、参数量（Params）等指标，重点关注 Macro-F1 以评估不平衡场景鲁棒性。
